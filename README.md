# 🏆 Panini Tracker 2026

Panini Tracker 2026 este o aplicație Android realizată cu React Native + Expo pentru gestionarea albumului Panini FIFA World Cup 2026.

## Funcționalități

- ✅ Album
- ✅ Căutare
- ✅ Filtrare
- ✅ SQLite
- 🚧 Statistici
- 🚧 Duplicates
- 🚧 Import Excel
- 🚧 Backup

## Tehnologii

- React Native
- Expo SDK 57
- Expo SQLite
- React Navigation
- Expo Vector Icons

## Instalare

```bash
npm install
npx expo start
